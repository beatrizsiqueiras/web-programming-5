8 - Explique sobre o DOM no Javascript e qual seu principal papel

O DOM (Document Object Model) é uma interface de programação no JavaScript que representa a estrutura de um documento HTML ou XML. Seu papel principal é permitir a interação e manipulação dos elementos da página web de forma dinâmica, possibilitando acessar, modificar e estilizar elementos, além de responder a eventos do usuário.

10 - Qual a ordem de precedência em caso de aparecimento de operadores lógicos em uma mesma expressão? Como é feito a leitura?

A ordem de precedência dos operadores lógicos em uma expressão em JavaScript é a seguinte:
NOT (!)
AND (&&)
OR (||)
A leitura de uma expressão com operadores lógicos é feita da esquerda para a direita. No caso do operador NOT, ele é aplicado primeiro a um valor ou expressão. A avaliação dos operadores AND e OR é feita da esquerda para a direita. Em expressões mais complexas, é possível usar parênteses para controlar a ordem de avaliação dos operadores lógicos, definindo assim a precedência desejada.

11 - Qual a diferença entre os operadores `==`e `===`?

O operador == faz uma comparação de valor, permitindo a coerção de tipos, enquanto o operador === faz uma comparação de valor e tipo, sem coerção de tipos.
